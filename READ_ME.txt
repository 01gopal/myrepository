Do the following steps:
1. Download the zip from the repository -> https://github.com/01gopal/myrepository
2. Unzip the folder.
3. To run default ATM, execute runATM_Default.bat -> Go to myrepository-master and double click "runATM_Default.bat"
4. To manually initialize the ATM execute runATM_ManualInit.bat -> Go to myrepository-master and double click "runATM_ManualInit.bat"
5. To run test cases execute runATMTests.bat -> Go to myrepository-master and double click "runATMTests.bat"

PRE-REQUST for Execution:
1. Maven(3.5)
2. JDK-1.8
3. M2_HOME and JAVA_HOME should be set up in EnvionmentVariable and PATH.
